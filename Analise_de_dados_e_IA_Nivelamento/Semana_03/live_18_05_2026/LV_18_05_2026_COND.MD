## 2 . LOOP de repetição
Na ultima aula visualizamos o trecho do código no qual era solicitado ao usuário a inserção de de uma informação __(SIM ou NÃO)__, porém nesse código não era obrigado que o usuário escolhesse algumas dessas opções, então para tal iremos iniciar essa validação com laço `While`
```python
idosos = input("Você é idose ?\n (sim/não) -> ")
while idosos != "sim" and idosos != "não":
    print("Você deve digitar apenas sim ou não!!")
    idosos = input("Você é idose ?\n (sim/não) -> ")

cartao = input("Você possui cartão de idoso? (sim/não): ")

if idosos == "sim" and cartao == "sim":
    print("Pode estacionar!")
else:
    print("Procure outra vaga")
```
Nessa condição de verificação o laço será repetido enquanto não houve uma condição de veracidade atendida, no caso como estamos combinando com `sim` e `não` na tabela verdade que havíamos visto anteriormente, uma condição com `AND` é atendida desde que __AO MENOS UMA DAS CONDIÇÕES__ sejam verdadeiras
>PS: para melhora a validação podemos adicionar a função `.lower()` para que a variável que foi enviada seja transformada em minusculo.

> __Tabela Verdade `OR`__
> |       |     |       |           |
> | ----- | --- | ----- | --------- |
> | TRUE  | OR  | FALSE | __TRUE__  |
> | FALSE | OR  | TRUE  | __TRUE__  |
> | TRUE  | OR  | TRUE  | __TRUE__  |
> | FALSE | OR  | FALSE | __FALSE__ |
> ----
> __Tabela Verdade `AND`__
> |       |     |       |           |
> | ----- | --- | ----- | --------- |
> | TRUE  | AND | FALSE | __FALSE__ |
> | FALSE | AND | TRUE  | __FALSE__ |
> | TRUE  | AND | TRUE  | __TRUE__  |
> | FALSE | AND | FALSE | __FALSE__ |

Também e importante se atentar que para `loops` do tipo `while` por exemplo  dentro do bloco do código, a variável de validação tenha um trecho de modificação do valor para que atenda a condição.

## 3. Listas
O comando para lista ou atribuição de listas em python podemos seguir a lógica de por exemplo:
```py
vogais = ["a", "e", "i", "o", "u"]
```
Nessa parte atribuímos uma lista de caracteres e essa lista é feita com `[]` com os valores dentro desses colchetes. 
para substituir a verificação de `or's` e `and's` que foram feitas anteriormente, podemos realizar o comando de verificação com o comando __`in`__ esse comando percorre um valor de uma lista e fazer a verificação conforme exemplo:  
```py

vogais = ["a", "e", "i", "o", "u"]

letra = input("Digite uma letra: ")

if letra in vogais:
    print(f"{letra} é vogal")
else:
    print(f"{letra} não é uma vogal")

```
outra maneira de utilização de listas para simplificar o processo de verificações pode-se feito como :  
```py
estado_civil = input(
    "Você é solteiro/a(s), casado/a(c), divorciado/a(d) ou viúvo/a(v)\n->"
)
while estado_civil.lower(set) not in ["s", "c", "d", "v"]:
    print("ERRO")
    estado_civil = input(
        "Você é solteiro/a(s), casado/a(c), divorciado/a(d) ou viúvo/a(v)\n->"
    )
```

## 4. Laço de repetição FOR
Para exemplificação de um laço for temos:  
```py
nomes = ["Gabi", "Danilo", "Sabino", "Aceloti", "Toninho", "Leninha"]

for nome in nomes:
    print(f"{nome} é professor/a")

```
Ao utilizar o código acima o `Python` realiza o print de cada um dos nomes presentes na lista por print, ou seja ele percorre a lista como um todo e quebra cada um para o nome variável nome, e esses valores presentes na lista serão impressos enquanto houver valores na lista.
